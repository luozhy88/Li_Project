# ThyGlycoPortal: 甲状腺癌糖基化交互分析平台

## 项目概述

ThyGlycoPortal 是一个专注于甲状腺癌糖基化研究的交互式分析平台。本项目基于 R Shiny 构建，**所有展示数据均为已发表文献的真实汇总统计量或公开数据库的真实数据**，不包含任何模拟数据。

数据来源包括：
- Zhang ZJ et al. Front Oncol. 2021 (PMID: 34476207) — IgG N-糖组诊断数据
- Kudelka MR et al. Cancer Med. 2023 (PMID: 36437732) — 血清复发预测数据
- PTMC Nomogram. Front Oncol. 2022 (PMC9497917) — 淋巴结转移风险数据
- Bones J et al. Cancers. 2018 (PMC11727208) — **TCGA RNAseq 糖基因表达数据**
- Wu et al. Front Endocrinol. 2021 (PMC8267918) — 血浆糖组学数据

## 快速启动

### 前置条件

- Python 3.x
- R >= 4.3 (需安装 shiny, shinydashboard, DBI, RSQLite, dplyr, ggplot2, plotly, DT, pROC, tidyr, readr)

### 一键启动 Shiny 应用

```bash
# 如果数据库尚未创建，先运行 Phase 1 数据准备
python3 05_create_database.py
python3 06_import_glycomics_data.py

# 启动 R Shiny 应用 (默认端口 3838)
Rscript 09_run_shiny.R

# 或使用自定义端口
Rscript 09_run_shiny.R 8080
```

启动后，在浏览器中访问：`http://localhost:3838`

### 在 R 中直接启动

```r
shiny::runApp("app.R", port = 3838)
```

## 功能模块

### 1. Overview (概览)
- 数据库统计仪表盘（研究数、糖链数、标志物数、酶数）
- 研究年份分布、癌种分布、方法学分布可视化
- 生物标志物 AUC 比较图

### 2. Literature (文献)
- 交互式文献表格，支持搜索、排序、筛选
- 按年份、癌种、方法学、样本类型多维过滤

### 3. Biomarkers (生物标志物)
- 标志物性能对比（AUC、敏感性、特异性）
- 详细信息面板

### 4. Glycan Browser (糖链浏览器)
- 24种核心N-糖链结构浏览
- **跨研究比较**：查看同一糖链在不同文献中的变化趋势

### 5. Enzyme Network (酶-糖链网络)
- 8种关键糖基转移酶信息查询
- 酶调控的糖链关联网络

### 6. TCGA Glycogene Expression (TCGA糖基因表达)
- **真实 TCGA RNAseq 数据** (Bones et al. 2018, n=20 paired Normal vs PTC)
- 展示真实的 median RPKM、fold change 和显著性
- 支持上传用户自己的表达矩阵进行对比

### 7. Diagnostic Tool (诊断工具)
- **IgG BN 诊断评分**（基于 Zhang 2021，AUC=0.920）
- **血清 G0F:G1F 复发风险预测**（基于 Kudelka 2023，AUC=0.82）

### 8. Nomogram (列线图)
- PTMC 淋巴结转移风险概率计算
- 动态仪表盘可视化

### 9. Data Upload (数据上传)
- 上传用户自己的糖组学 CSV 数据
- 自动生成糖谱轮廓图和组间比较图

## 注意事项

1. **数据真实性**：本平台**不使用任何模拟数据**。TCGA 糖基因表达数据来自 Bones et al. 2018 对 TCGA RNAseq 的分析（n=20 paired）；诊断工具和列线图基于文献报告的真实性能参数（AUC、cutoff、敏感性、特异性）。

2. **数据层级**：由于原始作者通常不公开个体水平原始数据，当前数据库主要包含：
   - 文献汇总的元数据和变化趋势
   - 真实的均值±标准差、中位数、AUC、p值
   - 真实的 TCGA 配对样本 median RPKM 值

3. **扩展建议**：
   - 从 GlycoPOST 下载公开质谱原始数据（GPST000197, GPST000495）
   - 从 GEO 下载甲状腺癌基因表达数据集
   - 使用 TCGAbiolinks 批量下载完整 TCGA-THCA 数据
