📍 穴位识别

使用预定义的穴位词典，包括中英文名称和经络代码。 在文献标题和摘要中进行字符串匹配。

覆盖穴位：足三里(ST36)、太冲(LR3)、百会(GV20)、 合谷(LI4)、三阴交(SP6)、内关(PC6)、风池(GB20)、 曲池(LI11)、阳陵泉(GB34)、关元(CV4) 等 80+ 个穴位

🧬 基因识别

采用两层策略：先匹配已知基因词典，再使用正则模式捕获候选基因符号， 最后通过停用词表过滤非基因词汇。

已知基因词典：TNF、IL6、IL10、AKT、MAPK、MTOR、 NLRP3、BCL2、STAT3、WNT 等 200+ 个针灸研究高频基因
正则模式：\b[A-Z][A-Za-z0-9]{1,14}\b

🏥 疾病/表型识别

基于 MeSH 词表和常见疾病关键词列表，在摘要中进行关键词匹配。 覆盖炎症、疼痛、抑郁、帕金森、肿瘤、糖尿病等主要疾病类型。

🔗 关联分析算法
基于文献共现分析的穴位-基因关联挖掘

1. 共现频率 (Co-occurrence Frequency)

两个实体（如某个穴位和某个基因）在同一篇文献中同时出现的次数。 共现频率越高，表明两者在研究中被同时关注的程度越高。

CoFreq(A, B) = |{文章 i : 实体 A ∈ 文章 i 且 实体 B ∈ 文章 i}|

2. Jaccard 相似度

用于衡量两个实体关联的相对强度。取值范围 [0, 1]， 值越大表示关联越强。考虑了各自的总文献数，比单纯共现频率更具可比性。

Jaccard(A, B) = |A ∩ B| / |A ∪ B| = CoFreq(A, B) / (Freq(A) + Freq(B) - CoFreq(A, B))
举例：若穴位 ST36 出现在 96 篇文献中，基因 TNF 出现在 55 篇文献中， 两者共现 21 篇，则 Jaccard = 21 / (96 + 55 - 21) = 21 / 130 = 0.1615

3. 支持文献 (Evidence Articles)

对于每一对穴位-基因关联，列出所有同时提及两者的原始文献。 每篇文献提供 PMID、标题、期刊、年份等信息，便于追溯原始证据。

4. 网络分析指标

指标 定义 意义
度数 (Degree) 节点的连接数 度数越大，该实体关联的其他实体越多
节点大小 映射度数 视觉上突出核心穴位/基因
边宽度 映射共现次数 连线越粗，关联越强
